## Overview

This document outlines everything that will be included in the Content Management System (CMS) that Bytes Platform will create and configure specifically for The Benavente Group's website at benaventegroup.com.

## Your Website Pages

Your website consists of five main pages. Each page has dedicated content sections that you will be able to manage directly through the CMS.

### 1. Home Page

The main landing page for The Benavente Group. It introduces the firm and directs visitors to the key areas of the site.

**Home Page — What You Can Edit**

- ✓ Hero background video (the drone footage displayed at the top of the page)
- ✓ Main headline text (e.g. "Real Estate Valuation & Consultancy")
- ✓ Scrolling services ticker — the list of services that animates across the screen
- ✓ "About the Firm" section — introductory paragraph and body text
- ✓ Statistics counters — Years of Experience, Assignments, Pacific Regions, Property Types
- ✓ Featured portfolio previews shown on the homepage (up to 5 project images)
- ✓ Comments section — a slider where you can add, modify, or remove client testimonials

### 2. About Page

This page introduces your firm's story, mission, values, regional coverage, and the full team.

**About Page — What You Can Edit**

- ✓ Page banner image (the aerial coastline photo displayed at the top)
- ✓ Hero headline and subtitle text
- ✓ "Our Story" section — firm background paragraph and the three stat numbers
- ✓ Mission statement headline and supporting text
- ✓ Core values — edit titles and descriptions for Accuracy, Integrity, Timeliness, and Local Knowledge
- ✓ Regional coverage section — photos and labels for each island/region
- ✓ Team member profiles — add, remove, or update name, title, photo, and CV file for each person

### 3. Portfolio / Gallery Page

This page showcases your completed projects organized by property type. Visitors can browse by category — Retail, Industrial, Hotel & Hospitality, and more.

**Portfolio Page — What You Can Edit**

- ✓ Add new project photos to any existing category
- ✓ Remove outdated or unwanted images from any category
- ✓ Create new property categories as your portfolio grows
- ✓ Update category names and display labels

### 4. Blog Page

Your blog allows you to publish market insights, analysis, and industry articles — which also helps improve your website's visibility on Google (SEO).

**Blog Page — What You Can Edit**

- ✓ Write and publish new blog posts
- ✓ Edit or update any existing published posts
- ✓ Delete posts you no longer want displayed

### 5. Contact Page

The contact page allows prospective and existing clients to get in touch with your firm directly.

**Contact Page — What You Can Edit**

- ✓ Office address displayed on the page
- ✓ Phone number displayed on the page
- ✓ Email address displayed on the page
- ✓ View incoming messages submitted through the contact form

## How It Works

The CMS will be straightforward and designed to be used without any technical background. Making an update will be as simple as:

- Log in to your private admin panel using your credentials
- Navigate to the section you want to update (e.g. Team Members, Blog Posts, Gallery)
- Make your changes — edit text, upload a new image, publish a new post, etc.
- Save or publish — changes go live on benaventegroup.com immediately
- No code required. No developers needed for day-to-day content updates

Bytes Platform will walk you and your team through the CMS during a brief onboarding session to make sure everything is clear and comfortable before handoff.
